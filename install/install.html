Installation

This section provides more detailed information about installing Lazyearth. This includes details about Lazyearth's prerequisites as well as Lazyearth's required and optional dependencies. A cross platform distribution for earth data analysis and scientfic computing.

If you don't have Python yet and want the simplest way to get started,I recommend you use the Anaconda Distribution. It includes Python, Numpy and many other commonly used packages for scientific computing and data science.

  • Use Anaconda Navigator to manage your packages
  • For writing and executing code, use notebooks in JupyterLab for exploratory and interactive computing
  • Visual Studio Code (VScode) for writing scripts and packages.

Lazyearth can be installed with pip on Windown and Linux, or from source. For more detailed instructions, consult our Python and Lazyearth installation guide below

Lazyearth is officially supported (and continuously tested) on CPython versions 3.7 and later. It’s possible that Lazyearth does work on other versions of Python, but no guarantees or support are provided.


Python & Environment manager

A virtual environment will prevent possible installation conflicts with other Python versions and packages. It's optional but strongly recommend Create a new virtual environment for your Lazyearth project rather than install in the base environment. You can Install the 64bit version of Python 3, for instance from https://www.python.org.

conda create --name lazy-env python=3.7

Copy

You will have to do this step from the current directory every time you start a new terminal. This sets up the environment.

conda activate lazy-env

Copy

Your terminal should now preface the path with something like (lazy), indicating that the lazy environment is active. If it doesn’t say that, the virtual environment is not active and the following won’t work.

(lazy) C:\Users\lazyman>Desktop>

Installing from PyPI

The best way to install lazyearth is with the pip tool (which is what python uses to install packages). Note, this comes with python in recent versions. you can install Lazyearth with:

pip install lazyearth
