Genguasian

objearth.common.genguasian()

It's a quick way to create a guasian fileter

Parameter : x_size : int

Size of guasian in x axis

y_size : int

Size of guasian in y axis

Return :

 ndarray

Example

Create guasian size (5,10)

>>> from lazyearth.common import objearth as oe 
>>> gaussian_img = oe.genguasian(50,50)
>>> oe.plotshow(gaussian_img)
