bandcombination


objearth.bandcombination()

used to create combination with 3 bands image (xarray.Dataset/numpy.ndarray)

Parameter : band1 : xarray.DataArray/numpy.ndarray

                     band1 image


                    band2 : xarray.DataArray/numpy.ndarray

                     band2 image


                    band3 : xarray.DataArray/numpy.ndarray

                     band3 image


                    bright : int (optional)

                     image brightness


Return : numpy stacked array
