Genimg

objearth.common.genimg()

Used for creating an array to use as an image for testing.

Parameter : Size : list,optional

Size of array

range : list,optional

Range of value

nan : int,optional

Number of nan

inf : int,optional

Number of infinity

Return :

 ndarray

Example1

create simple array

>>> from lazyearth.common import objearth as oe
>>> oe.genimg()
>>> array([[ 0.6793952 , -0.81620819],
        [-0.62274242, -0.38527724]])

create array with special size and value

>>> oe.genimg(size=[5,5],range=[-10,10])
>>> array([[-8.6014487  , -4.1402669 ,  4.13306626, -0.35697446,  8.11139776],
           [-4.26807551 , -4.8475852 , -7.69419216, -1.04074379,  4.09444996],
           [-9.65702999 ,  4.80955952, -4.54168647,  3.23779609, -8.60125623],
           [ 2.70531363 , -7.71183384,  0.42417152, -9.47133212,  9.26505475],
           [-1.49992575 ,  8.62291597,  1.7773813 ,  1.27451696,  6.91072804]])

create array with nan and inf

>>> oe.genimg(size=[3,3],range=[-10,10],nan=2,inf=2)
>>> array([[inf, 8.92091758, 0.614487 ],
           [-3.896195 , 6.83857102,nan ],
           [nan, 3.42085332,inf]])
