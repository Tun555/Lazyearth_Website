objearth.xxx()

used for joining multiple band image (Red Green Blue)

Parameter : Red : ndarray

                     Red band image


                    Green : ndarray

                     Green band image


                    Blue : ndarray

                     Blue band image


                    bright : int,optional

                     image brightness


Return : ndarray
References

1 gdal "Opening the File", https://gdal.org/tutorials/raster_api_tut.html

Example

Open image file

>>> from lazyearth import objearth as oe
>>> rgb = oe.band_combination(red,green,blue)
>>> oe.plotshow(rgb)
