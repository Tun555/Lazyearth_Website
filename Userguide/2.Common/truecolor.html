Truecolor

objearth.common.truecolor()

used to create truc coloe image from xarray dataset (1 time)

Parameter : Dataset : xarray.Dataset (time:1)

xarray dataset that have to include red,green,blue band with 1 time

Return :

 numpy.ndarray

Create truecolor image
