Bare soil index (BSI)

index.BSI(xarray.dataset)

Use to calculate Bare soil index (BSI). The band than nessessary for function is swir,blue,red,nir bands

Parameter : DataArray : xarray.dataset

Xarray dataset that contain swir,blue,red,nir bands

Return :

  xarray.dataset

References

1 Bare Soil Index (BSI), https://giscrack.com/list-of-spectral-indices-for-sentinel-and-landsat/
