Normalized difference water index (NDWI)

index.NDWI(xarray.dataset)

Use to calculate Normalized difference water index (NDWI). The band than nessessary for function is swir,nir bands

Parameter : DataArray : xarray.dataset

Return :

  xarray.dataset

References

1 Normalized difference water index, https://en.wikipedia.org/wiki/Normalized_difference_water_index
